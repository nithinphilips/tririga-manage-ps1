# Open-TririgaRDP

## SYNOPSIS
Opens an RDP client connection to the TRIRIGA server

## SYNTAX

```
Open-TririgaRDP [-environment] <String> [[-instance] <String>] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION
Launches the Microsoft Remote Desktop Connection tool with the server name pre-filled.
