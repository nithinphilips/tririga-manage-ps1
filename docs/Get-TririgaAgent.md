# Get-TririgaAgent

## SYNOPSIS
Gets TRIRIGA Agents configuration

## SYNTAX

```
Get-TririgaAgent [-environment] <String> [-instance <String>] [[-agent] <String>] [-running] [-notRunning]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Gets TRIRIGA Agents configuration

Uses the /api/v1/admin/agent/status method
